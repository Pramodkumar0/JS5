  • Functional scope is name as local scope, the only difference is that the variable is defined inside a function.

6.Explain passed by value and passed by reference?

Ans:
passed by value:
  • Pass by value is primitive datatype.
  • Because premitive data types are starts with numbers,booleans,strings
  • The method parameter values are copied to another variable and then the copied object is passed, that's why it's called pass by value
         var a = 10; 
        var b = a + 20;
        console.log(b)

passed by reference:
  • pass by reference is the non-primitive data type.
  • Non-primitive data types are objects and arrays
  • An reference to the actual parameter is passed to the method, that's why it's called pass by reference.
    let obj1 = {
        name: "hello",
        age: 22
    }
    obj1.age = 18
    let obj2 ={
        name : "hello",
        age: 24
    }
    console.log(obj1)
    console.log(obj2)
